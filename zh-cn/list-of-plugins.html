插件

升级插件的方式

cooking remove vue
cooking import vue

通过 extends 我们可以引入额外的配置和依赖，目前提供的插件有：

  • vue - 提供 vue 相关配置
  • react - 提供 react 相关配置

Lint 相关

  • lint - 使用 ESLint

CSS 相关

  • sass - sass 配置
  • less - less 配置 by webjyh
  • postcss(1.0 弃用) - postcss-loader, 内置 cssnext 等
  • saladcss(1.0 弃用) - 更好用的 CSS 预处理插件 postcss-salad

测试相关

  • karma - karma 相关依赖

更多插件可以在 https://github.com/cookingjs 找到，同时欢迎你来开发新的插件。
