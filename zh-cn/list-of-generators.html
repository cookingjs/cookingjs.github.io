cooking

脚手架

升级脚手架的方式是先要卸载掉原有的，再安装新的

cooking remove vue -t
cooking import vue -t

项目类

  • vue - 基础的 vue 项目的脚手架，支持 Vue 1 和 2
  • react - 基础的 react 项目的脚手架
  • static - 基础页面项目

工具类

  • plugin - 开发 cooking 插件的脚手架
  • generator - 开发 cooking 脚手架的脚手架
